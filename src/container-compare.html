Bootstrap Container 실제 속성 확인

커스텀 CSS 없이, 오직 Bootstrap 순수 클래스로만 시각화한 화면입니다.

1. .container (고정폭/반응형)

현재 이 파란색 영역이 .container의 실제 크기입니다.

  • width: 기본 100%로 시작하지만, 화면이 넓어지면 특정 크기(Breakpoint)에서 폭이 딱 고정됩니다.
  • margin: 고정폭이 된 순간부터 남는 공간을 좌우로 똑같이 나눠 가져가므로 화면 중앙에 배치됩니다. (양옆의 회색 여백이 마진입니다.)

2. .container-fluid (전체폭)

현재 이 빨간색 영역이 .container-fluid의 실제 크기입니다.

  • width: 화면 크기와 상관없이 언제나 무조건 100% 전체를 차지합니다.
  • margin: 좌우 여백이 남지 않기 때문에 화면 왼쪽 끝과 오른쪽 끝에 항상 밀착되어 있습니다.
